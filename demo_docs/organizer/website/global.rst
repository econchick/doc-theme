Setup your Machine
------------------

To make *any* changes, let's setup your machine first.

1. `Fork`_ the `pyladies/pyladies`_ repository to your own (or your local PyLadies) GitHub account.
2. Locally clone **your** fork of the repository.
3. Create a new `virtualenv`_.  Within that virtualenv, run ``pip install -r requirements.txt``.
